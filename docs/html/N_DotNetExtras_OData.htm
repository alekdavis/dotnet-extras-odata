      DotNetExtras.OData Namespace

      Provides classes for parsing and validating OData filter expressions.

      Classes

      ODataFilterNode Implements a node of the binary tree holding OData operator or operand use in a filter query.
      ODataFilterNodeCount Defines the minimum and maximum numbers.
      ODataFilterRules Defines collections of rules that can be applied to the operators and properties used in an OData filter expression.
      ODataFilterTreeT Implements a binary tree that holds the OData filter expression.
      ODataFilterValidatorT Validates OData filter expression for the given data type and the set of rules.
